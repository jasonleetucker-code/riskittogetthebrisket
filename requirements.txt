# ── Runtime dependencies ──────────────────────────────────────────────
#
# Single source of truth for the Python runtime.  Every environment
# (local dev, CI, production) installs from THIS file via
# ``pip install -r requirements.txt``.  Test/dev-only dependencies
# live in ``requirements-dev.txt`` which chains in this file via
# ``-r requirements.txt`` so there is no duplication.
#
# Versions use compatible-release (``~=``) pins so patch/minor
# bumps flow in but a surprise major-version bump cannot silently
# break CI or production.  To lock exact versions for reproducibility,
# run ``pip freeze > requirements.lock.txt`` after a clean install.
#
# Do NOT add test/dev-only dependencies here.  Add them to
# ``requirements-dev.txt`` instead.
